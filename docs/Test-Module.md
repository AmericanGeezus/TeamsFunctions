# Test-Module

## SYNOPSIS
Tests whether a Module is loaded

## SYNTAX

```
Test-Module [-Module] <String> [<CommonParameters>]
```

## DESCRIPTION
Tests whether a specific Module is loaded

## EXAMPLES

### EXAMPLE 1
```
Test-Module -Module ModuleName
```

Will Return $TRUE if the Module 'ModuleName' is loaded

## PARAMETERS
